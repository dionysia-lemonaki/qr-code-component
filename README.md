### Useful resources

- [Variable fonts guide](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_fonts/Variable_fonts_guide) - MDN's documentation on variable fonts helped me correctly write the `@font-face` declaration.
- [The Complete Guide to Centering in CSS](https://moderncss.dev/complete-guide-to-centering-in-css/) - Stephanie Eckles’ thorough guide outlines various ways to center a div. I used the 'XY Grid Solution,' which she calls the easiest and most modern approach.
- [How TO - Responsive Images](https://www.w3schools.com/howto/howto_css_image_responsive.asp) - This W3Schools resource helped me make the QR image responsive by combining `height: auto` with `max-width: 100%`.
